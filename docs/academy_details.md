# Academy Details

This document outlines the overall structure, systems, and academic rules of The Learners Academy.

## Academic Structure

### Term System
- **Cycle**: 3 months per term.
- **Seasons**: Terms are based on seasons:
  - Spring
  - Summer
  - Autumn
  - Winter
- **Transition**: A new term starts immediately after the previous 3-month cycle.

### Class Duration
- **Time**: Each class is strictly 1 hour long.

## Grading & Assessment

### Assessment Rules
- **Frequency**: Two tests are taken per term.
- **Marks**: Each test is graded out of 100 marks.
